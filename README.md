# AlphaFin LTE

한국 주식 시장 환경에서 동작 가능한 경량형 AlphaFin 스타일 금융 분석 시스템 구축 프로젝트.

이 프로젝트는 원본 AlphaFin 논문의 전체 구조를 완벽히 재현하는 것이 아니라,
무료 데이터 소스와 실험 가능한 수준의 경량 아키텍처를 기반으로 다음 목표를 달성하는 것을 목표로 한다.

- 한국 금융 데이터 수집 자동화
- RAG 기반 금융 정보 검색
- LLM 기반 주식 분석 흐름 구축
- 반복 가능한 금융 AI 실험 환경 구축
- AI 협업(Codex / Cursor) 기반 개발 구조 구축

---

# 프로젝트 목표

본 프로젝트의 핵심 목표는 다음과 같다.

## 1. 한국 환경 기반 AlphaFin LTE 구축

다음 데이터를 활용한다.

- 한국 주식 시세
- OpenDART 공시
- 기업 재무 데이터
- 뉴스 데이터
- 시장 이벤트 데이터

이를 기반으로:

- 검색 가능한 금융 지식 저장소
- RAG 기반 검색 구조
- LLM 기반 분석 흐름

을 구축한다.

---

## 2. AI 협업 최적화 개발 환경 구축

이 프로젝트는 단순 코드 저장소가 아니라,
AI 협업 기반 개발 환경 구축 자체도 중요한 목표로 한다.

다음 도구를 활용한다.

- ChatGPT
- Codex
- Cursor

프로젝트는 하네스 엔지니어링(Harness Engineering) 방식으로 구성된다.

---

# 프로젝트 범위

현재 포함 범위:

- pykrx 기반 시세 수집
- OpenDART 기반 공시 수집
- 뉴스 데이터 수집
- 문서 저장 구조
- 문서 전처리
- Chunking
- Vector Search
- RAG 프로토타입
- LLM 분석 흐름
- 간단한 평가 및 실험

현재 제외 범위:

- 실거래 자동화
- 초고빈도 거래(HFT)
- 대규모 분산 인프라
- MSA
- 원본 AlphaFin 완전 재현
- 대규모 GPU 클러스터
- 프로덕션 트레이딩 시스템
