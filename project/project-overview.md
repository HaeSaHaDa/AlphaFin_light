# AlphaFin LTE 프로젝트 개요

## 프로젝트 소개

AlphaFin LTE는
한국 주식 시장 환경에서 동작 가능한
경량형 금융 AI 연구 프로젝트이다.

이 프로젝트는 원본 AlphaFin 논문의 전체 구조를
완벽하게 재현하는 것을 목표로 하지 않는다.

대신 다음 목표를 중심으로 한다.

- 한국 금융 데이터 수집
- 검색 가능한 금융 정보 구조 구축
- RAG 기반 검색 시스템 구축
- LLM 기반 금융 분석 흐름 구축
- 반복 가능한 금융 AI 실험 환경 구축
- AI 협업 기반 개발 구조 구축

---

# 프로젝트 배경

기존 금융 AI 연구는 다음 문제를 가진다.

- 대규모 인프라 요구
- 복잡한 데이터 파이프라인
- 높은 구현 비용
- 재현 어려움
- 실험 구조 관리 어려움

특히 AlphaFin류 연구는:

- 금융 데이터 수집
- Retrieval
- Memory 구조
- LLM 분석
- 평가 시스템

등이 복합적으로 연결되어 있어
개인 또는 소규모 환경에서 구현하기 어렵다.

AlphaFin LTE는 이러한 문제를 완화하기 위해
경량형 구조를 목표로 한다.

---

# 핵심 목표

현재 프로젝트의 핵심 목표는 다음과 같다.

## 1. 한국 금융 데이터 기반 실험 환경 구축

다음 데이터를 활용한다.

- 한국 주식 시세
- OpenDART 공시
- 뉴스 데이터
- 기업 정보

이를 기반으로
검색 및 분석 가능한 금융 데이터 환경을 구축한다.

---

## 2. 경량형 RAG 금융 분석 구조 구축

다음을 목표로 한다.

- 문서 저장
- Chunking
- Embedding
- Retrieval
- Context Assembly
- LLM 기반 분석

대규모 분산 인프라 대신
실험 가능한 수준의 단순 구조를 우선한다.

---

## 3. AI 협업 기반 개발 구조 구축

이 프로젝트는 단순 금융 프로젝트가 아니라,
AI 협업 최적화 구조 구축 자체도 중요한 목표로 한다.

다음 도구를 활용한다.

- ChatGPT
- Codex
- Cursor

프로젝트는 문서 기반 하네스 엔지니어링 방식으로 진행된다.

---

# 프로젝트 특징

## 문서 우선 개발

구현보다 문서를 먼저 정의한다.

우선 순위:

1. 문서 정의
2. TASK 정의
3. 구조 정의
4. 구현

---

## 작은 단위 개발

모든 구현은 작은 단위로 유지한다.

예:

- 작은 TASK
- 작은 함수
- 작은 모듈
- 작은 커밋

---

## 실험 가능한 구조 우선

현재 목표는
완전한 금융 플랫폼 구축이 아니라
반복 가능한 실험 환경 구축이다.

따라서 다음을 우선한다.

- 단순성
- 추적 가능성
- 디버깅 가능성
- 재현 가능성

---

# 현재 구현 범위

현재 포함 범위:

- pykrx 기반 시세 수집
- OpenDART 기반 공시 수집
- 뉴스 수집
- 문서 저장
- RAG 프로토타입
- LLM 분석 흐름
- 간단한 평가 및 실험

현재 제외 범위:

- 실거래 자동화
- HFT
- 대규모 분산 시스템
- 초대형 벡터 인프라
- 원본 AlphaFin 완전 재현

---
